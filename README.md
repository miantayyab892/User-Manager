# 🛠️ Linux User Management Tool

A professional, colorful, and interactive Bash script to manage system users in Linux (Kali Linux, Ubuntu, etc.). This tool allows you to create, delete, and list users with a single click.

## ✨ Features
* **Create User**: Easily add new system users with home directories.
* **Delete User**: Remove users and their data completely.
* **View Users**: List the most recent users added to the system.
* **Colorful UI**: Uses ANSI escape codes for a professional terminal experience.
